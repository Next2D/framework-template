=============

[![CodeQL](https://github.com/Next2D/framework-template/actions/workflows/github-code-scanning/codeql/badge.svg?branch=main)](https://github.com/Next2D/framework-template/actions/workflows/github-code-scanning/codeql)

[![license](https://img.shields.io/github/license/Next2D/framework-template)](https://github.com/Next2D/framework-template/blob/main/LICENSE)

Next2D Framework default JavaScript template.\
Create Next2D apps with no build configuration.

## Quick Start

```sh
npx create-next2d-app sample-app --template @next2d/framework-template
cd sample-app
npm start
```
